REPL

REPL is the Read-Eval-Print loop.

In order to get one, run one of the following:

Operating SystemProgram to Run
UN*X5D (the GUI), T5D (the shell).
Windows5D (the GUI), T5D (the shell).

The T5D shell looks like this:

5D Version 0.6.7 - Copyright (C) 2011 Danny Milosavljevic et al.
This program comes with ABSOLUTELY NO WARRANTY.
It is free software, and you are welcome to redistribute it under certain conditions. See /usr/share/doc/5d/copyright for details.
eval $

In order to import modules, use import!.

REPL.import! "<name>"

import! will:

  • ensure that the given module is loaded
  • import the symbols of the module that are on the whitelist and not on the blacklist into the current namespace

(In fact, import! is shorthand for a combination of define! (or let) and requireModule).

In order to define some variable, use define!.

REPL.define! '<name> (<body>)"

Example:

REPL.define! 'f (\x x*x)
return! (f 2) => 4
