Why?

You might ask "Why another language?".

I, of all people, would use an existing language for what I'm trying to do if possible.

So far I've tried several dozens of languages and none come even close.

Goals

  • Exploratory Programming: get results as soon as you ask, the "ask questions, get answers" property.
  • Higher-order Programming: if I have to write another for loop, I'm going to scream.
  • Numeric Tower: if using numbers, have them behave as in ordinary mathematics (the one everyone on the planet learns in high school).
  • Reflectiveness: Allow finding out of what exists by asking the system.
  • Succintness: as simple as possible but no simpler.

What?

So far, there exist:

  • the 5D Language Specification
  • an interpreter for the language, see the 5D User Manual for how to run the interpreter.
  • documentation.

Where?

5D Main Site

How?

For how to use the language, see 5D Tutorial and 5D Programming Manual.

For how to use the standard libraries, see 5D Library Documentation.
